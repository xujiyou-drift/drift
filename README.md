# Drift

> 目标是做一个 Kubernetes 之上的 "CDH"
>
> 实际上是一个 Kubernetes Operator
>
> 可以通过界面部署和管理大数据组件
>
> 支持持久化储存

## 支持的组件

- [x] ZooKeeper 3.6.1
- [ ] Kafka
- [ ] Yarn
- [ ] HDFS
- [ ] HBase
- [ ] Hive
- [ ] Spark
- [ ] Flink
